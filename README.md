A curated collection of interactive data analytics dashboards built using Power BI, Python, Excel, Tableau, and SQL.
This repository showcases end-to-end skills including data cleaning, exploratory data analysis (EDA), feature engineering, KPI design, and insight-driven visualization.
Each dashboard focuses on solving real-world problems such as performance analysis, customer churn, sales trends, and operational insights, making it suitable for internship and entry-level data analyst roles.
